\begin{abstract}
This paper presents a rigorous comparative analysis of Byzantine Fault Tolerant (BFT) consensus algorithms for securing Electric Vehicle (EV) energy trading on IEEE 33-bus distribution systems. We evaluate eight consensus protocols across four evolutionary groups---Baselines (OM($m$), Classic PBFT), Enterprise Consortium (IBFT~2.0, QBFT), Node-Clustered Energy PBFT (CE-PBFT, G-PBFT, SV-PBFT), and Next-Generation Sub-Second frameworks (Tower~BFT, RVR Consensus)---through the lens of Sheikh et al.'s probabilistic attack model (Eq.~14--21) extended with the Static-Temporal Security Framework (STSF). \textbf{Algorithms:} CE-PBFT \cite{ding2024}, G-PBFT \cite{xu2025}, SV-PBFT \cite{suo2025}.

\textbf{Algorithmic Parameters:} These models partition the massive sensor grid into small, isolated committees ($c$) using spectral clustering, geohashing, or vehicle-to-vehicle reputation. Cluster sizes typically range from $c = 20$ to $c = 30$, with latency $L \approx 400$--$800$ ms.

\textbf{Key Modification to STSF:} The communication channel count changes from global to localized:
\begin{equation}
k_1^{cluster} = \frac{c(c-1)}{2} \quad \text{vs} \quad k_1^{global} = \frac{n_{sen}(n_{sen}-1)}{2}
\end{equation}

For CE-PBFT with $c = 25$: $k_1^{cluster} = 300$ vs $k_1^{global} = 7{,}426{,}681$. This dramatically reduces $P_{CAb}$, protecting against communication path exploits.

\textbf{Temporal Results:}
\begin{align}
N_{CE} &= 16, \quad P_{temporal}^{CE} \approx 0.077 \\
N_{G} &= 13, \quad P_{temporal}^{G} \approx 0.063 \\
N_{SV} &= 10, \quad P_{temporal}^{SV} \approx 0.049
\end{align}

\textbf{Result:} $P_{secure}$ ranges from $0.923$ to $0.951$, approaching but not reaching the $0.98$ target threshold.
